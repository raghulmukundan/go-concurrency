Introduction to Go Concurrency

Welcome to my collection of notes on Go concurrency! This website documents my learning journey through Go's powerful concurrency features.

Go's approach to concurrency is unique and powerful, making it easier to write programs that get the most out of multicore and networked machines.

Goroutines

Goroutines are lightweight threads managed by the Go runtime. They're incredibly efficient and easy to create.

Basic Example

func main() {
    go sayHello("World")
    time.Sleep(100 * time.Millisecond)
}

func sayHello(name string) {
    fmt.Printf("Hello, %s!\n", name)
}

Channels

Channels are the pipes that connect concurrent goroutines. You can send values into channels from one goroutine and receive those values in another goroutine.

Basic Channel Usage

func main() {
    messages := make(chan string)
    
    go func() {
        messages <- "ping"
    }()
    
    msg := <-messages
    fmt.Println(msg)
}
